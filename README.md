# 📚 Bibliotech — Digital Library Interface

Este projeto é o resultado de um desafio prático proposto pelo **Professor Gabriel** durante o curso de **Análise e Desenvolvimento de Sistemas (ADS)** no **SENAI**. O objetivo principal foi consolidar conhecimentos em **HTML5** e **CSS3**, explorando a criação de uma interface moderna.

## 🎯 O Desafio
A proposta era criar o front-end de uma biblioteca digital (Bibliotech) que foi criada pelo professor, para que pudessemos aprender mais sobre web tentando replicar os elementos da biblioteca...
## 🛠️ Tecnologias e Conceitos Aplicados

Durante o desenvolvimento, foquei em aplicar as melhores práticas aprendidas em aula:

- **Flexbox Layout:** Utilizado em toda a aplicação para garantir o alinhamento preciso dos elementos e a distribuição dos cards.
- **Estilização Avançada:** Manipulação de cores, tipografia, efeitos de hover e bordas personalizadas para criar uma identidade visual única.
- **Componentização Visual:** Criação de padrões para botões, inputs e cards de livros, facilitando a manutenção do código.

## 🧩 Estrutura do Projeto

O layout foi dividido em seções estratégicas:
1.  **Navegação:** Menu clean com sistema de login para membros.
2.  **Hero Section:** Área de destaque com busca centralizada e tipografia impactante.
3.  **New Arrivals:** Vitrine de livros com sistema de cards interativos.
4.  **Upcoming Events:** Calendário de eventos com layout de datas diferenciado.
5.  **Service Center:** Seção de suporte ao usuário com ícones e descrições.
6.  **Footer:** Rodapé informativo com divisões por colunas e redes sociais.
